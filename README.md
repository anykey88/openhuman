# OpenHuman

你的个人 AI 超级智能。私密、简单、极其强大。

OpenHuman 是一个开源的 AI 助手平台，运行在你的本地电脑上。它连接 118+ 个服务（Gmail、日历、WhatsApp、Notion 等），能帮你处理邮件、安排日程、管理文档、回答问题。

## 特性

- 🧠 **多模型支持** — 接入 OpenAI、Anthropic、Google Gemini、本地 Ollama 等
- 🔌 **118+ 服务集成** — Gmail、日历、WhatsApp、Notion、Slack、GitHub 等
- 🔒 **本地优先** — 数据存在你的电脑上，隐私安全
- 🎤 **语音输入** — 说话就能操作
- 📅 **会议转录** — 自动记录和总结会议
- 🧠 **记忆树** — AI 记住你的对话和偏好
- 🌍 **多语言** — 支持中文、英文、日文、韩文等
